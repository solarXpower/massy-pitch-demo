# Websites Workspace

## Purpose
Improve existing commercial retail store websites — UI/UX redesigns, performance, SEO, accessibility, and conversion optimization.

## Structure
Each subfolder is a separate retail client/store:
- `Massy/` — Massy retail chain

## Approach for Each Client
1. Audit the existing website (design, performance, SEO, accessibility, UX)
2. Identify quick wins vs. larger improvements
3. Implement changes with before/after comparisons where possible

## Conventions
- Keep original site structure intact; improve incrementally
- Document findings per client in a `NOTES.md` inside each client folder
- Place any shared components or templates in `_shared/`

## Priorities (in order)
1. Performance (load speed, Core Web Vitals)
2. Mobile responsiveness
3. UX / conversion rate improvements
4. SEO
5. Accessibility (WCAG 2.1 AA)
